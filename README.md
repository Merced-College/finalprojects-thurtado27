Abstract
This program simulates a fast food ordering system where users can view a menu, place orders, and process them in a queue system. It demonstrates how real-world ordering systems work using data structures and algorithms.

Data Structures Used
ArrayList: stores menu items and order items
Queue: stores customer orders in FIFO order
LinkedList (Queue implementation): handles order processing

Algorithms
Search: finds menu items by index
Sum: calculates total order price
Queue processing: handles orders in FIFO order

Big-O Analysis
Search: O(n)
Total calculation: O(n)
Queue processing: O(1)

Reflection
One challenge was managing orders in correct sequence. This was solved using a queue, which ensures first-in-first-out processing. In future versions, I would add payment options and order customization.
