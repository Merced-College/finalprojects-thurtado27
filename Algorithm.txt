FAST FOOD SYSTEM - ALGORITHMS

1. Calculate Total (Recursive)
---------------------------------
Class: Order.java
Method: calculateTotalRecursive()

Algorithm:
- Input: List of MenuItem objects in the order
- Base Case: If index < 0, return 0
- Recursive Case: Return price of item at current index + recursiveTotal(index - 1)
- Output: Total cost of the order

Usage:
- Called in Receipt.printReceipt() to calculate the total cost of the order
- Demonstrates recursion

Time Complexity: O(n), where n = number of items in the order


2. Process Order Queue
---------------------------------
Class: OrderQueue.java
Method: processOrder()

Algorithm:
- Input: Queue of orders
- Action: Remove the first order from the queue (FIFO)
- Output: The removed order

Usage:
- Called in Main.java when processing the next order
- Implements queue behavior

Time Complexity: O(1)


3. Track Order Statistics
---------------------------------
Class: OrderStats.java
Method: addOrder()

Algorithm:
- Input: Order object
- For each MenuItem in the order:
    - If item name exists in HashMap, increment count
    - Else, add item name to HashMap with count = 1
- Output: Updated HashMap of item counts

Usage:
- Called after processing an order to track total quantities of each item ordered
- Demonstrates usage of HashMap as a hash table

Time Complexity: O(m), where m = number of items in the order


4. Menu Item Selection
---------------------------------
Class: Menu.java
Method: getItem(int index)

Algorithm:
- Input: Integer index
- Check if index is valid (0 <= index < size of menu)
- If valid, return the corresponding MenuItem
- Else, print error and return null

Usage:
- Called when placing an order to select items from the menu

Time Complexity: O(1)
